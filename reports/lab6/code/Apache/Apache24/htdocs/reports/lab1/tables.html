ТЕОРЕТИЧНА ЧАСТИНА

ТЕГ TABLE

Атрибути

  • align - визначає вирівнювання таблиці
  • background - задає фоновий малюнок в таблиці
  • bgcolor - колір фону таблиці
  • border - товщина рамки в пікселях
  • bordercolor - колір рамки
  • cellpadding - відступ від рамки до вмісту комірки
  • cellspacing - відстань між комірками
  • cols - кількість колонок в таблиці
  • frame - повідомляє браузеру, як відображати межі навколо таблиці
  • height - висота таблиці
  • rules - повідомляє браузеру, як відображати межі між комірками
  • summary - короткий опис таблиці
  • width - ширина таблиці

ТЕГ TR

Атрибути

  • align - визначає вирівнювання вмісту комірок по горизонталі
  • bgcolor - колір фону комірок
  • bordercolor - колір рамки
  • char - вирівнює вміст комірок відносно заданого символу
  • charoff - зміщує вміст комірок відносно заданого символу
  • valign - вирівнює вміст комірок по вертикалі

ТЕГ TD

Атрибути

  • abbr - короткий опис вмісту комірки
  • align - визначає вирівнювання вмісту комірки по горизонталі
  • axis - групує комірки, що пов'язані схожою інформацією
  • background - задає фоновий малюнок в комірці
  • bgcolor - колір фона комірки
  • bordercolor - колір рамки
  • char - вирівнює вміст комірки відносно заданого символу
  • charoff - зміщує вміст комірки відносно заданого символу
  • colspan - об'єднує горизонтальні комірки
  • headers - дозволяє об'эднати комырки ыз заголовком
  • height - висота комірки
  • nowrap - забороняє переносити рядки
  • rowspan - об'єднує вертикальні комірки
  • scope - дозволяє об'єднати комірки простої таблиці із заголовками в мовних браузерах
  • valign - вирівнює вміст комірки по вертикалі
  • width - ширина комірки
