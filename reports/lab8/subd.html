Командне завдання


Для створення таблиць та бази даних використали Entity Framework Core, а точніше підхід code first, коли спочатку створюються сутності, а на основі них таблиці.


Робота з СУБД здійснена через написання класів та використання Entity Framework Core для створення бд в MSSQL. Для правильної роботи ORM Entity Framework у класах були прописані навігаційні властивості(для встановлення зв'язків) та прописані атрибути DataAnnotation для деяких властивостей.


Наведемо власне список таблиць нашої Бази даних.

Наведемо власне файл з таблицями нашої Бази даних.


Наведемо список Сервісів та їх інтерфейсів, які використовують власне для отримання інформації з бази даних, пошуку інформації в базі даних, додаванню/видаленню даних, загалом - взаємодією розробника із базою даних.

Наведемо приклад запитів для отримання різного роду інформації з декількох або однієї таблиці.
