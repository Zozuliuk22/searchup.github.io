ТЕОРЕТИЧНА ЧАСТИНА

ТЕГ FORM

Атрибути

  • accept-charset - встановклює кодування, в якому сервер може приймати та обробляти дані
  • action - адреса документа або програми, яка обробляє дані форми
  • autocomplete - вмикає автозаповнення полів форми
  • enctype - спосіб кодування даних форми
  • method - метод протоколу HTTP
  • name - ім'я форми
  • novalidate - відміняє вбудовану перевірку даних форми на коректність вводу
  • target - ім'я вікна або фрейму, куди обробник буде завантажувати повертаємий результат

ТЕГ INPUT

Атрибути

  • accept - встановлює фльтр на типи файлів, котрі ми можемо надіслати через поле загрузки файлів
  • accesskey - перехід до елементу за допомогою комбінації клавіш
  • align - визначає вирівнювання зображення
  • alt - альтернативний текст для кнопки із зображенням
  • autocomplete - вмикає або вимикає автозаповнення
  • autofocus - встановлює фокус в поле форми
  • border - товщина рамки навколо зображення
  • checked - попередньо активований перемикач або прапорець
  • disabled - блокує доступ и змінення елемента
  • fоrm - пов'язує поле з формою за її ідентифікатором
  • formaction - визначає адресу обробника форми
  • formenctype - встановлює спосіб кодування даних форми при їх відправці на сервер
  • formmethod - повідомляє браузеру яким методом варто передавати дані форми на сервер
  • formnovalidate - відміняє вбудовану перевірку даних на коректність
  • formtarget - визначає вікно або фреймв яке буде завантажуватись результат, повернений обробником форми
  • list - вказує на список варіантів, котрі можна вибирати при вводі тексту
  • max - верхнє значення для вводу числа або дати
  • maxlength - максимальна кількість символів дозволених в тексті
  • min - нижнє значення для вводу числа або дати
  • multiple - дозволяє загрузити декілька файлів одночасно
  • name - ім'я поля, призначеного для того, щоб обробник форми міг його ідентифікувати
  • pattern - встановлює шаблон вводу
  • placeholder - виводить текст-підказку
  • readonly - встановлює, що поле не може змінюватись користувачем
  • required - обов'язкове для заповнення поле
  • size - ширина текстового поля
  • src - адреса графічного файлу для поля із зображенням
  • step - крок збільшення для числових полів
  • tabindex - визначає порядок переходу між елементами за допомогою клавіші Tab
  • type - повідомляє браузеру, до якого типу відноситься елемент форми
  • value - значення елемента
